# 详细设计：界面子系统

## 1. 概述

界面子系统负责系统的整体界面框架和布局，包括顶部工具栏、侧边栏、内容展示区、手机模拟器区域、抽屉面板和布局管理。界面子系统为其他两个子系统提供容器和导航支持。

## 2. 模块图

![界面子系统_模块图](images/界面子系统_模块图.png)

### 模块职责

| 模块 | 职责 |
|------|------|
| 顶部工具栏模块 | 应用标题展示、全局菜单（侧边栏折叠、抽屉开关、重置、导出） |
| 侧边栏模块 | 需求分类列表展示，支持分类切换导航 |
| 内容展示区模块 | 承载需求收集子系统的表格渲染组件 |
| 模拟器区域模块 | 承载交互式需求收集子系统的手机模拟器组件 |
| 抽屉面板模块 | 展示组件和布局列表，支持拖拽到模拟器 |
| 布局管理模块 | 管理三栏布局的尺寸、折叠和显隐 |

### 模块间关系

| 提供方 | 消费方 | 说明 |
|--------|--------|------|
| 顶部工具栏模块 | 侧边栏模块 | 控制侧边栏折叠/展开 |
| 顶部工具栏模块 | 抽屉面板模块 | 呼出/关闭抽屉面板 |
| 顶部工具栏模块 | 模拟器区域模块 | 控制模拟器显隐 |
| 布局管理模块 | 侧边栏/内容区/模拟器区 | 管理三栏布局尺寸 |
| 侧边栏模块 | 内容展示区模块 | 分类切换触发内容刷新 |
| 抽屉面板模块 | 模拟器区域模块 | 拖拽目标指向模拟器 |

## 4. 类详细说明

### 4.1 顶部工具栏模块

#### AppHeader

应用顶部栏组件。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| title | string | 应用标题 |
| menuItems | MenuItem[] | 菜单项列表 |
| render() | VNode | 渲染顶部栏 |

#### ToolBar

工具栏操作按钮组。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| onToggleSidebar() | void | 切换侧边栏折叠/展开 |
| onToggleDrawer() | void | 切换抽屉面板开关 |
| onReset() | void | 重置模拟器 |
| onExport() | void | 导出需求结果 |

#### MenuItem

菜单项数据类。

| 属性 | 类型 | 说明 |
|------|------|------|
| label | string | 菜单项文字 |
| icon | string | 图标标识 |
| action | string | 对应操作名称 |

### 4.2 侧边栏模块

#### SideBar

侧边栏容器组件。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| collapsed | boolean | 是否折叠 |
| selectedCategoryId | string | 当前选中分类ID |
| toggle() | void | 切换折叠状态 |
| setCollapsed(collapsed) | void | 设置折叠状态 |
| render() | VNode | 渲染侧边栏 |

#### CategoryList

分类列表组件。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| categories | Category[] | 分类数据列表 |
| activeId | string | 当前激活分类ID |
| setCategories(categories) | void | 设置分类数据 |
| setActive(id) | void | 设置激活分类 |
| onCategoryChange(callback) | void | 注册分类切换回调 |
| render() | VNode | 渲染分类列表 |

### 4.3 内容展示区模块

#### ContentArea

中间内容区域容器，承载需求收集子系统的表格组件。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| currentView | string | 当前视图名称 |
| viewData | object | 视图数据 |
| setView(viewName, data?) | void | 切换视图和数据 |
| render() | VNode | 渲染内容区域 |

### 4.4 模拟器区域模块

#### SimulatorArea

右侧手机模拟器容器，承载交互式需求收集子系统的模拟器组件。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| visible | boolean | 是否可见 |
| simulatorComponent | Component | 模拟器组件实例 |
| setVisible(visible) | void | 设置可见性 |
| mountSimulator(component) | void | 挂载模拟器组件 |
| unmountSimulator() | void | 卸载模拟器组件 |
| render() | VNode | 渲染模拟器区域 |

### 4.5 抽屉面板模块

#### DrawerPanel

抽屉面板容器，包含组件和布局两个标签页。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| open | boolean | 是否打开 |
| activeTab | string | 当前标签页（component/layout） |
| toggle() | void | 切换开关状态 |
| setActiveTab(tab) | void | 切换标签页 |
| close() | void | 关闭抽屉 |
| render() | VNode | 渲染抽屉面板 |

#### ComponentDrawer

组件选择抽屉页，展示组件列表并支持拖拽。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| components | ComponentInfo[] | 组件列表 |
| registry | ComponentRegistry | 组件注册表引用 |
| setComponents(components) | void | 设置组件数据 |
| render() | VNode | 渲染组件卡片列表（可拖拽） |

#### LayoutDrawer

布局选择抽屉页，展示布局列表并支持拖拽。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| layouts | LayoutInfo[] | 布局列表 |
| registry | LayoutRegistry | 布局注册表引用 |
| setLayouts(layouts) | void | 设置布局数据 |
| render() | VNode | 渲染布局卡片列表（可拖拽） |

### 4.6 布局管理模块

#### LayoutManager

三栏布局管理器，控制侧边栏、内容区、模拟器区域的尺寸和显隐。

| 属性/方法 | 类型 | 说明 |
|-----------|------|------|
| sidebarWidth | number | 侧边栏宽度 |
| contentWidth | number | 内容区宽度（自动计算） |
| simulatorWidth | number | 模拟器区域宽度 |
| simulatorVisible | boolean | 模拟器是否可见 |
| setSidebarWidth(width) | void | 设置侧边栏宽度 |
| toggleSimulator(visible) | void | 切换模拟器显隐 |
| getContentWidth() | number | 获取内容区宽度 |
| getLayoutConfig() | LayoutConfig | 获取完整布局配置 |

#### LayoutConfig

布局配置数据类。

| 属性 | 类型 | 说明 |
|------|------|------|
| sidebarWidth | number | 侧边栏宽度（默认240px） |
| contentMinWidth | number | 内容区最小宽度（默认400px） |
| simulatorWidth | number | 模拟器区域宽度（默认420px） |
| simulatorVisible | boolean | 模拟器是否可见（默认true） |

```

|          |                           |                       |

| Category |  TableRenderer            |   PhoneFrame          |

|          |                           |                       |
|          |                           |                       |
+----------+---------------------------+-----------------------+

               [DrawerPanel] (浮动，从右侧滑出)
               +-----------------------+
               | [ComponentDrawer]     |
               | [LayoutDrawer]        |
               +-----------------------+
```

## 6. 模块对外接口

本子系统对外暴露 `I_Navigation` 和 `I_Layout` 两个接口：

| 对外接口 | 实现模块 | 实现类 |
|----------|----------|--------|
| I_Navigation.switchCategory | 侧边栏模块 | CategoryList.onCategoryChange |
| I_Navigation.toggleDrawer | 顶部工具栏模块 | ToolBar.onToggleDrawer |
| I_Layout.setSimulatorVisible | 布局管理模块 | LayoutManager.toggleSimulator |
| I_Layout.setContentData | 内容展示区模块 | ContentArea.setView |
