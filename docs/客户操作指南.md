# 需求收集工具 - 客户操作指南

## 1. 打开工具

在浏览器地址栏中输入工程师提供给你的完整访问地址，**必须包含端口号**。例如工程师告诉你访问地址是 `http://192.168.1.100:8000`，则在浏览器中输入：

```
http://192.168.1.100:8000
```

> **注意**：地址中的 `:8000` 是端口号，不能省略。请直接使用工程师给你的完整地址。

打开后你会看到如下界面：

```
+--------------------------------------------------------------+
|                    需求收集工具（顶部工具栏）                    |
+----------+---------------------------+-----------------------+
|          |                           |                       |
| 需求分类  |      问题表格              |    手机模拟器          |
|          |                           |                       |
| 基本信息  |  问题 | 目的 | 回答 | 备注 |   ┌─────────────┐    |
| 页面设计  |  ...  | ... | ... | ... |   │  模拟手机屏幕  │    |
| 商品页面  |                           |   │               │    |
| ...      |                           |   └─────────────┘    |
+----------+---------------------------+-----------------------+
```

## 2. 填写需求

### 2.1 选择需求分类

点击左侧的**分类名称**（如"基本信息"、"页面设计"），中间区域会显示该分类下的问题列表。

### 2.2 回答问题

问题表格有四列：**问题、目的、回答、备注**。你只需要在"回答"列中操作。

#### 文字输入

直接在输入框中打字即可。

```
问题: 您的企业名称是什么？
目的: 用于项目标题
回答: [请输入_______________]
备注: 请填写全称
```

#### 选项选择

勾选你需要的选项，可多选。如果看到"自定义"选项且旁边出现输入框，可以在其中输入你自己的内容。

```
问题: 您需要哪些功能？
目的: 确定开发范围
回答: ☑ 产品展示   ☐ 在线预约   ☑ 用户管理   ☑ 自定义 [___]
备注: 可多选
```

#### 交互式页面设计

点击**"打开模拟器设计"**按钮，右侧会出现一个手机模拟器。

```
问题: 请设计首页布局
目的: 了解页面偏好
回答: [打开模拟器设计] [保存设计结果]
备注: 拖动组件到模拟器中设计
```

## 3. 使用手机模拟器

### 3.1 打开组件面板

点击顶部工具栏的**"组件面板"**按钮，右侧会弹出一个面板，显示可用的组件和布局列表。

```
+------------------+
| [组件] [布局]     |    ← 切换标签页
+------------------+
| ■ 按钮           |    ← 可拖拽的组件
| ■ 文本           |
| ■ 图片           |
| ■ 输入框         |
| ■ 商品卡片       |
| ■ ...            |
+------------------+
```

### 3.2 添加组件

从面板中**拖动**一个组件卡片，**放到**手机模拟器屏幕中。组件会出现在模拟器内。

### 3.3 添加到布局容器内

如果模拟器中已有布局组件（如"纵向布局"、"横向布局"），你可以将组件拖入布局内部，组件会作为子组件显示。

### 3.4 删除组件

鼠标悬停在模拟器中的组件上，左上角会出现一个红色 **×** 按钮，点击即可删除。

### 3.5 保存设计结果

完成设计后，点击问题表格中的**"保存设计结果"**按钮。保存后会显示"已设计"标记。

### 3.6 查看默认布局

如果工程师提供了默认布局，打开模拟器时会自动加载。你可以在此基础上修改（添加、删除组件），也可以删除全部重新设计。

## 4. 工具栏操作

顶部工具栏提供以下按钮：

| 按钮 | 功能 |
|------|------|
| 收起/展开侧栏 | 隐藏或显示左侧分类列表 |
| 组件面板 | 打开或关闭组件/布局选择面板 |
| 隐藏/显示模拟器 | 隐藏或显示右侧手机模拟器 |
| 保存结果 | **将所有填写内容保存**（重要！） |

## 5. 保存你的需求

**填完所有问题后，请务必点击顶部工具栏的"保存结果"按钮。**

点击后会弹出"需求结果已保存"提示，表示保存成功。

> 每次点击保存会覆盖之前的结果。如果你需要修改某个回答，直接修改后再次点击"保存结果"即可。

## 6. 操作提示

- 回答会**自动记录**，切换分类不会丢失之前填写的内容
- 如果页面意外关闭，只要之前点击过"保存结果"，工程师仍然可以获取你上次保存的结果
- 手机模拟器中的设计需要点击**"保存设计结果"**才会关联到对应问题
- 如果你不确定某个问题怎么回答，可以参考"备注"列中的提示
- "目的"列说明了为什么问这个问题，帮助你理解工程师需要什么信息

## 7. 常见问题

**Q: 页面打不开**
A: 请确认网址是否正确，联系工程师确认服务是否正常运行。

**Q: 模拟器中看不到组件**
A: 点击顶部"组件面板"按钮打开面板，从面板中拖动组件到模拟器。

**Q: 组件拖不到模拟器里**
A: 确保拖动的是面板中的组件卡片，拖到模拟器的白色手机屏幕区域释放。

**Q: 之前填写的内容不见了**
A: 如果你之前点击过"保存结果"，工程师可以从服务器获取你的结果。请联系工程师。

**Q: 如何修改已提交的内容**
A: 直接在页面上修改，然后再次点击"保存结果"即可覆盖之前的结果。
