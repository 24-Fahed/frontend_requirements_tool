# 需求收集工具 - 客户操作指南

## 1. 打开工具

在浏览器中访问工程师提供的完整地址，例如：

```text
http://192.168.1.100:8000
```

打开后，你会看到三栏界面：

1. 左侧：需求分类
2. 中间：问题表格
3. 右侧：手机模拟器

## 2. 回答普通问题

### 2.1 文本问题

直接在输入框中填写内容即可。

### 2.2 选项问题

勾选需要的选项；如果题目允许“自定义”，可额外输入自己的内容。

## 3. 设计交互式页面

### 3.1 打开模拟器

在交互式问题所在行点击 **“打开模拟器设计”**。

如果工程师预先配置了默认布局，模拟器会自动加载默认页面结构。

### 3.2 打开组件面板

`v0.2.0` 起，组件面板不再由顶部工具栏控制。

请在手机模拟器顶部状态栏点击 **`Palette`** 按钮，打开组件和布局面板。

### 3.3 添加组件

在组件面板中：

1. 选择 `Components` 查看普通组件
2. 选择 `Layouts` 查看布局容器
3. 将卡片拖动到模拟器画布中

### 3.4 删除组件

鼠标悬停在画布中的组件上，左上角会显示删除按钮 `×`，点击即可删除。

### 3.5 预览效果

点击模拟器状态栏中的 **`Preview`** 按钮可切换到预览模式：

- 隐藏编辑边框和删除按钮
- 按真实页面方式显示组件
- 再次点击按钮可回到编辑模式

### 3.6 保存设计结果

完成设计后，点击问题行中的 **“保存设计结果”**。

保存成功后，问题旁会显示 **“已设计”** 标记。

## 4. 顶部工具栏说明

顶部工具栏保留以下全局操作：

| 按钮 | 作用 |
|------|------|
| 收起/展开侧栏 | 控制左侧分类栏 |
| 隐藏/显示模拟器 | 控制右侧模拟器区域 |
| 保存结果 | 保存全部需求结果 |

说明：

- 顶部工具栏不再提供组件面板入口
- 组件面板入口已迁移到模拟器内部状态栏

## 5. 保存全部需求

完成所有填写后，请点击顶部工具栏的 **“保存结果”**。

系统会提示保存成功，表示当前需求已写入结果文件。

## 6. 常见问题

**Q：为什么顶部工具栏里看不到组件面板按钮？**  
A：这是 `v0.2.0` 的新设计。请在模拟器状态栏点击 `Palette`。

**Q：组件拖不到模拟器里怎么办？**  
A：请确认先打开模拟器内部的 `Palette`，再从面板卡片拖到手机画布区域。

**Q：预览模式下为什么不能继续拖拽？**  
A：预览模式只用于查看效果。点击 `Edit` 返回编辑模式后再继续操作。
